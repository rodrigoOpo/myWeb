# 🚀 Portfolio - Fullstack Web Developer

Bienvenido a mi portafolio personal.  
Este proyecto está diseñado para mostrar mis habilidades, proyectos y enfoque como **desarrollador web fullstack**, especializado en la creación de aplicaciones modernas, rápidas y escalables.

## 👨‍💻 Sobre mí

Soy **desarrollador web fullstack** y trabajo principalmente con:

- **Frontend:** Next.js, React, JavaScript / TypeScript
- **Backend:** FastAPI, Python
- **APIs:** REST APIs
- **Bases de datos:** PostgreSQL 
- **Herramientas:** Git, GitHub, Docker, Stripe

Mi objetivo es construir experiencias web limpias, funcionales y orientadas tanto al usuario como al negocio.
